```text
롤플레잉 비디오 게임에서 플레이언느 세계를 탐험하고 퀘스트를 완료하고 새로운 기술을 배우는 가상의 캐릭터의 역할을 맡는다. 
스킬 트리는 캐릭터의 성장을 진행하는 시스템 중 하나로 스킬의 계층 구조를 나타낸다.

이러한 시스템의 주요 어려움은 플레이어가 특정 새로운 스킬을 배우고 싶을 때마다 먼저 배워야 하는 또 다른 스킬이 있다는 것이다. 
이 규칙의 예외로는 조건 없이(먼저 배워야하는 스킬 없이) 가장 배우기 쉬운 스킬도 있다. 
"스킬 트리"라는 이름은 이러한 종속성이 가장 쉬운 스킬이 루트에 있는 트리 구조를 생성한다는 사실에서 파생된다.

한 번 스킬을 배우면 캐릭터가 항상 보유하게 되므로 다른 스킬을 필요로 하는 경우 다시 배울 필요가 없다. 
예를 들어 스킬 1에 스킬 0이 필요하고 스킬 2에 스킬 1이 필요하고 스킬 3에 스킬 1이 필요한 경우 스킬 2를 배우려면 플레이어는 2, 1, 0의 세 가지 스킬을 배워야 한다. 
스킬 3을 배우려면 전제 조건인 스킬 1과 0이 이미 충족되었기 때문에 스킬 3을 하나만 더 배우면 된다.

아래 그림1에서 빨간색은 배워야 할 스키를 낱낸고, 실선과 정점(노드)은 빨간색 잠금 해제하기 위해 배워야 하는 기술이며, 점선과 점선정점(점선노드)은 건너뛸 수 있는 것이다.
스킬 트리를 나타내는 N 정수의 배열 T가 제공도니다. K번째 스킬은 T[K]번째 스킬을 이미 학습한 경우에만 배울 수 있다. 
스킬 0은 루트이고 T[0] = 0이다. 위에서 설명한 예의 경우: T[0] = 0, T[1] = 0, T[2] = 1 및 T[3] = 1, 따라서 T = [0, 0, 1, 1].
```
